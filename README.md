## 🚀 Featured Projects

### 🏆 [AquaSync — Autonomous AI Buoy for Plastic Detection & Filtration](https://github.com/vijaymmp/aquasync)
> **Patented Innovation** · Indian Patent Application No. 202641073841

A solar-powered autonomous buoy that bridges **embedded hardware and real-time computer vision** to
tackle plastic pollution in rivers, lakes, ponds, and wells — without human intervention.

- 🧠 Engineered a **custom-trained YOLOv8** detection pipeline running against a live **ESP32-CAM** video feed to identify plastic debris — from large fragments down to micro-plastics — in real time
- 🔌 Architected the **hardware-software bridge**: microcontroller control logic (Arduino C++) translates model inference directly into physical actuation, closing the loop between "the model sees it" and "the buoy catches it"
- 🛰️ Designed **GPS-guided autonomous navigation** so the buoy patrols fixed routes and covers polluted zones efficiently, with zero manual piloting
- ⚙️ Built a **servo-controlled, vibration-assisted filtration net** to trap both large debris and fine plastic particles in slow-moving or still water
- 🌞 Delivered as a **low-cost, solar-powered, deployment-ready** system — engineered for real-world environmental use, not just a lab demo

`Arduino (C++)` `ESP32-CAM` `YOLOv8` `GPS Module` `Computer Vision`

### 📈 [The Data Time-Machine — GenAI-Augmented Forecasting Platform](https://github.com/vijaymmp/data-time-machine) *(Ongoing)*

An AI-powered analytics platform that turns raw historical data into **forward-looking, explainable
business narratives** — combining statistical forecasting with generative reasoning.

- 🔮 Combined **Meta Prophet time-series forecasting** with an **LLM-driven insight layer (Groq)** so every forecast ships with a plain-English "why," not just a chart
- 💬 Built a natural language **"What-If" simulation engine**, letting users query dynamic scenarios (e.g. *"What happens if demand drops 15% next quarter?"*) and get instant, model-backed answers
- 🖥️ Shipped an interactive **Streamlit** front end for exploring forecasts, scenarios, and narratives without touching code
- 🔗 Designed a **RESTful Flask API** backend to decouple the forecasting/LLM engine from the UI, making the platform extensible to other data sources and front ends
- 🧩 The core idea: analytics shouldn't just show *what happened* — it should **simulate what could happen next**, in language a non-technical stakeholder can act on

`Python` `Flask` `Meta Prophet` `Groq LLM` `Streamlit` `REST API`
